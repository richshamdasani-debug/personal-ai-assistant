### The 3 Core Pain Points

#### Pain 1: Setup Friction (73% of non-technical users never set up an AI agent)
- AutoGPT, AgentGPT, OpenClaw all require: Docker, API keys, terminal, server management
- Average setup time for a technical user: 4–6 hours
- Success rate for non-technical users: <5%
- **Result:** The people who need AI most can't access it

#### Pain 2: No Persistence or Proactivity (200M+ ChatGPT users, but avg session = 7 min)
- ChatGPT has no memory across sessions (or very limited)
- Users must re-explain context every single time
- ChatGPT only responds — it never acts or initiates
- **Result:** Powerful tool used for trivial tasks only

#### Pain 3: Living in the Wrong Place (apps, not messaging)
- 2 billion WhatsApp users, 800 million Telegram users
- People already live in these apps — all day, every day
- AI assistants require downloading new apps, learning new UIs
- **Result:** Low adoption, even lower retention
